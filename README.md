Recipe App 

A mobile application built with React Native (Expo) that allows users to explore Ethiopian recipes, view ingredients and preparation steps, and save favorite dishes for offline access.

Features

Search recipes by name or ingredient

View detailed recipe instructions and ingredients

Add and remove recipes from favorites

Offline access to saved favorite recipes

Clean and user-friendly UI

Tech Stack

React Native (Expo)

 TypeScript

TheMealDB API

AsyncStorage

React Navigation
